HTML


Kapitel 1 - Was ist HTML?


Mit HTML kann man das Grundgerüst einer Webseite coden. Bei HTML gibt es sogenannte Elemente.

Der öffnende Tag besteht aus einer öffnenden Spitzen Klammer, danach kommt das Zeichen und zuletzt die schliessende Spitze Klammer.

Der schliessende Tag besteht aus einer öffnenden Spitzen Klammer, gefolgt von einem Schrägstrich. Nach dem Schrägstrich kommt das Zeichen und wieder die schliessende spitze Klammer.

Zusätzlich gibt es bei einem Element auch noch einen Attributnamen und ein Attributwert.

Der Attributname zeigt, was für eine Art Zusatzinformation angegeben wird und der Attributwert ist die Information oder Einstellung des Attributs

Kapitel 2 - Text


Um bei einem Text Überschriften zumachen, gibt es folgende Elemente:

  • <h1>Überschrift 1</h1>

  • <h2>Überschrift 2</h2>

  • <h3>Überschrift 3</h3>

  • <h4>Überschrift 4</h4>

  • <h5>Überschrift 5</h5>
  • <h6>Überschrift 6</h6>

<h1></h1> ist die grösste Überschrift und wird immer kleiner.

Um einen Absatz zu erstellen, muss man den Text mit einem öffnenden <p> und einem schliessenden </p>-Tag umgeben

Man kann auch einen Text mit HTML gestalten, dies eignet sich bei einzelnen Worten.

<b>fett</b>
<i>kursiv</i>
<sup>hochgestellt</sup>
<sub>tiefgestellt</sub>
<strong>stark</strong>
<em>hervorheben</em>
<ins>unterstrichen</ins>
<s>durchgestrichen</s>

<br /> Umbruch
<hr />waggrechte Linie

Kapitel 3 - Listen


Bei HTML gibt es drei Arten von Listen: geordnete Listen, ungeordnete Listen und Definitionslisten.

Geordnete Listen:

Sie werden mit dem Element <ol> erstellt.
Jeder Eintrag wird zwischen dem öffnenden Tag <li> und dem schliessenden Tag </li> gestellt.
Die Aufzählungszeichen sind standardmässig nummeriert, diese Zeichen kann man aber im CSS bei list-style-type ändern.

<ol>

  1. <li>Gemüse</li>
  2. <li>Früchte</li>
  3. <li>Fleisch</li>
</ol>

Ungeordnete Listen:

Sie werden mit dem Element <ul> erstellt.
Jeder Eintrag wird zwischen dem öffnenden Tag <li> und dem schliessenden Tag </li> gestellt.
Die Aufzählungszeichen sind Punkte.

<ul>

  • <li>Gemüse</li>
  • <li>Früchte</li>
  • <li>Fleisch</li>
</ul>

Definitionslisten:

Sie werden mit dem Element <dl> erstellt.
Der Begriff, der definiert wird, steht zwischen dem öffnenden <dt> Tag und dem schliessenden </dt> Tag.
Die Definition wird von <dd> umschlossen.

<dl>

<dt>Subventionen</dt>
<dd>Zuschüsse vom Staat</dd>
</dl>
